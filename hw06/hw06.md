2.In your own words, describe the difference between the sep and collapse arguments to str\_c(). Answer: sep argument controls how strings are separated, using , or \* ect while collapse is to collapse a vector of strings into a single string. When we use collapse, it works on a string vector, that means we use c(). In other words, the sep argument is the string inserted between argugments to str\_c, while collapse is the string used to separate any elements of the character vector into a character vector of length one.

``` r
str_c("x", "y","z", sep = ", ")
```

    ## [1] "x, y, z"

``` r
str_c(c("x", "y", "z"), collapse = "  ")
```

    ## [1] "x  y  z"

``` r
turn<-function(x){
  y=""
  if(length(x)==1){
    y<-x[1]
  }
  else if(length(x)==2)
  {
    y<-str_c(x[1],' and ',x[2])
  }else{
    for(i in 2:length(x)-1)
    {
      y<-str_c(y," ",x[i])
      
    }
    y<-str_c(y," and ",x[length(x)])
  }

  print(y)
}
```

``` r
str_commasep <- function(x, sep = ", ", last = ", and ") {
  if (length(x) > 1) {
    str_c(str_c(x[-length(x)], collapse = sep),
```

1.  Write your own functions. In statistics, MLE(maximum likelihood estimates) is really important. Maximum likelihood estimates (MLEs) of model parameters are obtained by maximizing the likelihood. MLEs have nice asymptotic properties such as consistency, efficiency,and asymptotic normality. Sometimes, mle can not be easily obtained. In these cases, we can use numerical methods to get the mle such as bisection or Newton\_Raphson methods.

``` r
bisection <- function(f, a, b, n = 1000, tol = 1e-7) {
  # If the signs of the function at the evaluated points, a and b, stop the function and return message.
  if (f(a)*f(b)>0) {
    stop('finding root is fail')
  } 
  for (i in 1:n) {
    c <- (a + b) / 2 
    if ((f(c) == 0) || ((b - a) / 2) < tol) {
      return(c)
    }
    
    ifelse(sign(f(c)) == sign(f(a)), 
           a <- c,
           b <- c)
  }
  print('Too many iterations')
}
```

``` r
library(numDeriv)
newton.raphson <- function(f, a, b, tol = 1e-5, n = 1000) {
  x0 <- a # Set start value to supplied lower bound
  k <- n # Initialize for iteration results
  
  # Check the upper and lower bounds to see if approximations result in 0
  fa <- f(a)
  if (fa == 0.0) {
    return(a)
  }
  
  fb <- f(b)
  if (fb == 0.0) {
    return(b)
  }
  
  for (i in 1:n) {
    dx <- genD(func = f, x = x0)$D[1] # First-order derivative f'(x0)
    x1 <- x0 - (f(x0) / dx) # Calculate next value x1
    k[i] <- x1 
    if (abs(x1 - x0) < tol) {
      root.approx <- tail(k, n=1)
      res <- list('root approximation' = root.approx, 'iterations' = (k[1:5]))
      return(res)
    }

  x0<-x1
  }
}
newton.raphson(func,1,4.5)
```
